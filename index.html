01 WHY · [3/3]
현 학습용 게임 시장의 경향
종합 분석 결과
[1] 교사에게 편리한 서비스
• 낮은 진입 장벽 — PIN·QR 하나로 수업 중 즉시 활용
• 문항 구성 부담 제로 — 레디메이드 세트지 + 간편 추가 편집 가능
• 게이미피케이션 강화로 학생의 흥미·참여 유도
[2] 의미가 있는 서비스
• 수업 중 학생 진행 상황이 교사 화면에 자동 표시
• 수업 후 게임 결과가 자동 정리
• 틀린 문항 자동 추천 → 학생이 알아서 복습까지 연결
[3] 정교하고 풍성한 문항
• 레디메이드 세트지로 다양한 교과서·문제집 문항 제공
• 교사가 문항을 고르기만 하면 검증된 게임 수업 완성
• 메타정보 기반으로 학생 현황 자동 파악까지 지원
[4] 핵심 차별점 독립적 가치 + 결합 시너지
• 원하는 서비스만 따로 사용 가능
   (수업용 / 과제용 / 복습용 / 커뮤니티)
• 같이 쓰면 데이터·보상 연동으로 교사와 학생이 더 머무르고 싶은 서비스 생태계
기대 효과
교사가 덜 준비하고, 더 많이 파악하고, 계속 쓰게 된다

02 WHAT · [4/5]
교사에게 필요한 게임 서비스
정교하고 풍성한 문항
⛔ (구) CBS 일반 문항 → 게임 전환 어려움 · 일부 과목 세트지만 존재
▼
① 레디메이드 게임 문항 세트지
비상 교과서 + 타 교과서 + 문제집 — 교사는 선택만 하면 끝
STEP 1. 비상 교과서 세트지 제작
CBS 검증 문항을 게임용으로 변환하여 영어·수학·과학·사회 전과목 세트지 구축

STEP 2. 타 교과서·문제집 연계 — 교과서·문제집 구분 없이 게임 세트지로 즉시 활용
타 교과서 — 2022 성취기준 크로스 매핑 + 메타정보 자동 매핑
문제집(오투·완자 등) — 성취기준 기반 메타정보 자동 매핑

STEP 3. AI 유사 문항 추가 생성 — 부족한 영역도 즉시 보충
기존 세트지를 seed로 유사 문항 자동 생성 + 메타정보 자동 매핑
② 수업 후 자동 지원
모든 세트지 문항에 메타정보(지식요인·난이도) 탑재
학생 맞춤 진단 → 추천까지 자동 연결
